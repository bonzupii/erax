## Expected Behavior Summary:
1. Ligatures: Multi-char sequences render as single glyphs in monospace fonts
2. Box drawing: Perfect alignment, zero gaps between characters
3. Brahmi: Ancient script with combining marks, rare font required
4. Khmer: Complex subscript stacking, advanced shaping needed
5. Arabic: RTL, cursive joining, contextual forms
6. Tamil: Rounded script with vowel marks
7. Hebrew: RTL, block letters
8. Thai: No word spaces, tone marks above/below
9. CJK: EXACTLY 2 cells wide, full-width characters
10. Emoji: 2 cells wide, ZWJ sequences combine multiple emoji
11. BiDi: Mixed LTR/RTL text, cursor follows logical not visual order
12. Combining marks: Stack above/below base character
13. Control chars: Tab stops, non-breaking spaces, zero-width joiners
